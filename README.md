# Chicago Crime Data Analysis

This project explores crime patterns in Chicago using public crime and census datasets.

The goal was to understand how crime varies across time, location, and socio-economic conditions, and to practice working with real-world data.

## In this project, We analyzed:

- Crime trends over time (years, months, weekdays, time of day)
- Differences between crime types
- Police performance (arrest rates)
- Domestic crimes and crimes involving children
- Location-based patterns
- Spatial distribution using maps
- Relationship between crime and socio-economic factors

I also combined crime data with census data to explore how factors like income, unemployment, and hardship relate to crime patterns.

## Key insights

Some of the main findings include:

- Crime is not evenly distributed across time — certain days and periods show higher activity
- A small number of crime types (e.g. theft, battery) dominate overall crime
- Arrest rates remain relatively stable over time
- Domestic crimes involving children are strongly linked to domestic environments
- Some crime types show clear temporal and spatial patterns
- Socio-economic conditions appear to be associated with differences in crime levels across areas
